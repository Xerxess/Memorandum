

# 渲染模式

https://docs.godotengine.org/zh-cn/4.5/tutorials/rendering/renderers.html

## Forward+ 

最先进的渲染器，仅适合桌面平台，桌面平台默认使用。该渲染器使用 Vulkan、Direct3D 12 或 Metal 作为渲染驱动，使用 RenderingDevice 后端。

- 针对桌面进行开发
- 硬件较新，支持 Vulkan、Direct3D 12 或 Metal
- 开发 3D 游戏
- 使用最先进的渲染功能

## Mobile (移动)

功能较少，但渲染简单场景的速度更快。适用于移动平台和桌面平台。移动平台默认使用该渲染器。该渲染器使用 Vulkan、Direct3D 12 或 Metal 作为渲染驱动，使用 RenderingDevice 后端。

- 你在针对较新的移动设备、桌面 XR 或桌面进行开发。
- 硬件较新，支持 Vulkan、Direct3D 12 或 Metal。
- 开发 3D 游戏。
- 想要使用高级渲染功能，但还要考虑移动硬件的限制。

## Compatibility (兼容性)

在 OpenGL 3.3、OpenGL ES 3.0 和 WebGL 2.0 以上运行。最低级的图形后端，适合低端桌面和移动平台。在 Web 平台上默认使用。

- 为较旧的移动设备、较旧的桌面设备或独立的 XR 设备进行开发。Compatibility 渲染器支持的硬件最广泛。
- 在为 Web 平台开发。此时 Compatibility 是唯一的选择。
- 硬件较旧，不支持 Vulkan。此时 Compatibility 是唯一的选择。
- 开发 2D 游戏，或者不需要高级渲染功能的 3D 游戏。
- 想要在所有设备上获得最佳性能，不需要高级渲染功能。
