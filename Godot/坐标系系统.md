
# 坐标系系统

- 2D 空间 (Vector2)
  - X 轴：向右为正。
  - Y 轴：向下为正（注意：这与数学笛卡尔坐标系相反，但符合屏幕显示习惯）。
- 3D 空间 (Vector3)
  - X 轴：右 (Right)。
  - Y 轴：上 (Up)。
  - Z 轴：后 (Back)。在 Godot 中，物体的“正前方”默认指向 -Z 方向。

# 局部与全局定位

- position / rotation (局部)：相对于其直接父节点的偏移。如果父节点移动，子节点会随之移动，但其局部坐标保持不变。
- global_position / global_rotation (全局)：物体在整个游戏世界中的绝对位置。它计算了从场景根节点到该节点所有父级变换的叠加结果。
