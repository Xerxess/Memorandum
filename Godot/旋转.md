# 旋转

- Godot 的 2D 坐标系规定 0 弧度（\(0^{\circ }\)）指向右（X轴正向）。

## 旋转操作的方法

### @GlobalScope

- 方法
  - float deg_to_rad(deg: float) 将角度值转换为弧度值。
  - float rad_to_deg(rad: float) 将以弧度表示的角度转换为度。
  - float lerp_angle(from: float, to: float, weight: float) 通过 0.0 和 1.0 之间的 weight 值，在两个角度（以弧度为单位）之间进行线性插值。
  - float rotate_toward(from: float, to: float, delta: float) 将 from 朝 to 旋转，旋转的量为 delta。不会越过 to。

### Node2D

- 属性
  - float global_rotation 全局旋转，单位为弧度。
  - float global_rotation_degrees 辅助属性，用于按度数访问 global_rotation 而不是弧度数。
  - float rotation [默认： 0.0] 旋转，单位为弧度，相对于该节点的父节点。
  - float rotation_degrees 辅助属性，用于按度数访问 rotation 而不是弧度数。
- 方法
  - float get_angle_to(point: Vector2) 返回该节点和 point 之间的夹角，单位为弧度。
  - void look_at(point: Vector2) 旋转该节点，使其局部 X 轴的正方向指向 point，该参数应使用全局坐标。
  - void rotate(radians: float) 从节点的当前旋转开始，以弧度为单位，对节点进行旋转。

### Vector2

- 方法
  - float angle() 返回该向量与 X 轴正方向的夹角，单位为弧度。X 轴正方向为 (1, 0) 向量。
  - float angle_to(to: Vector2)  返回与给定向量的带符号夹角，单位为弧度。
  - float angle_to_point(to: Vector2) 返回连接两点的直线与 X 轴之间的夹角，单位为弧度。
  - Vector2 rotated(angle: float) 返回将这个向量旋转 angle 的结果（单位为弧度）。

## demo

```goscript
# 转向鼠标当前位置
look_at(get_global_mouse_position())
```
