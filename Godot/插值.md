

# 插值

插值就是在两个数值（起点 \(A\) 和终点 \(B\)）之间，根据一个百分比 \(t\)（\(0\) 到 \(1\) 之间），计算出中间某个位置的过程。

```text
以恒定速度将一个值转换为另一个值的插值被称为“线性”。
线性插值，指的是下面的公式。
公式：interpolation = A + (B - A) * t
例子： 起点是 0，终点是 100。如果 t=0.5（走了一半），结果就是 50。
```
