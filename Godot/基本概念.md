# 基本概念

## Texture 纹理

简单来说就是贴在 3D 模型或 2D 矩形上的“皮肤”或“图片数据”

## Material 材质

材质 (Material) 决定了物体表面如何与光线互动以及如何最终显示在屏幕上。

- 2D
  - CanvasItemMaterial (系统内置材质)
  - Blend Mode (混合模式)
    - Mix：普通渲染。
    - Add：发光效果（如火焰、激光）。
    - Subtract：减色效果。
    - Mul：正片叠底（常用于阴影）。
    - Light Mode (光照模式)：决定物体是否受 Light2D（灯光节点）的影响。可以设置为“仅受光照”、“不受光照”或“普通”。
  - ShaderMaterial (自定义着色器材质)
    - 想实现酷炫的特效（如：消融、流光、水波纹、灰度化），就需要用到它
