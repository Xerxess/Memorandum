# 闭包

- 闭包是一种函数，它可以"记住"定义它的作用域中的变量，即使在该作用域执行完毕后，这些变量仍然可以被访问。

```gdscript
# 闭包的基本概念
func create_counter():
    var count = 0
    
    # 这个函数就是闭包，它可以访问外部变量 count
    func increment():
        count += 1
        return count
    
    return increment

# 使用闭包
var counter1 = create_counter()
print(counter1.call())  # 输出: 1
print(counter1.call())  # 输出: 2
print(counter1.call())  # 输出: 3

# 创建另一个独立的计数器
var counter2 = create_counter()
print(counter2.call())  # 输出: 1 (独立的状态)
```
