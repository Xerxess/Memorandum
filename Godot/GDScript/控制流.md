# 控制流

## if

```gdscript
if (expression):
    statement(s)
elif (expression):
    statement(s)
else:
    statement(s)
```

## while

```gdscript
while (expression):
    statement(s)
```

## for

```gdscript
# 迭代数组
for x in [5, 7, 11]:
    statement

var names = ["John", "Marta", "Samantha", "Jimmy"]
for name: String in names:
    print(name)

# 字典
var dict = {"a": 0, "b": 1, "c": 2}
for i in dict:
    print(dict[i]) # Prints 0, then 1, then 2.
```

## match

相当于在许多其他语言中出现的 switch 语句

- 熟悉 switch 语句的人提供的速成
  - 将 switch 替换为 match
  - 删除 case
  - 删除 break
  - 将 default 替换为单个下划线_

```gdscript
match <test value>:
    <pattern(s)>:
        <block>
    <pattern(s)> when <pattern guard>:
        <block>
    <...>
```
