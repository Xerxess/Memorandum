# dataset 设置（获取）节点数据

* 由于小程序没dom节点，所有无法通过节点获得节点内容，通过data-* 设置获取

```html
<text data-value='test'>test</text>
```

```js
click(event) {
			let value = event.target.dataset.value;
		}
```

# canvasContent 的获取
