# uni-app

uni-app 是一个使用 Vue.js 开发跨平台应用的前端框架

注意：
* vue页面在App端是被系统的webview渲染

兼容平台：

* ios
* android
* h5
* 微信 app
* 支付宝 app
* 百度 app
* 头条 app

有点:

* 跨平台
* 提供平台api 调用，支持sdk
* 性能优秀
* 支付weex
* npm
* 易学习，成本低

功能：

* app 平台
    * 内置组件
    * 扩展组建/第三方插件
    * 第三方sdk(如微信支付，百度地图)
    * html5plus 
    * native.js
    * uni ios sdk
* 小程序
    * 内置组件
    * 小程序的专用api
* h5
    * h5专用api
