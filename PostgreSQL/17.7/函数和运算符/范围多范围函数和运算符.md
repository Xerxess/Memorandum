# 常用运算符（判断关系）

- anyrange @> anyrange → boolean 第一个范围是否包含第二个范围？
- anyrange @> anyelement → boolean 该范围是否包含该元素？
- anyrange <@ anyrange → boolean 第一个范围是否包含在第二个范围之内？
- anyelement <@ anyrange → boolean 该元素是否包含在指定范围内？
- anyrange && anyrange → boolean 这些范围是否有重叠，也就是说，是否有任何共同元素？
- anyrange << anyrange → boolean 第一个范围是否严格位于第二个范围的左侧？
- anyrange >> anyrange → boolean 第一个范围是否严格位于第二个范围的右侧？
- anyrange &< anyrange → boolean 第一个范围是否没有延伸到第二个范围的右侧？
- anyrange &> anyrange → boolean  第一个范围是否没有延伸到第二个范围的左侧？
- anyrange -|- anyrange → boolean  这两个范围相邻吗？
- anyrange + anyrange → anyrange 计算范围的并集。范围必须重叠或相邻，以便并集为单个范围（但请参阅 range_merge() ）。
- anyrange * anyrange → anyrange 计算两个范围的交集。
- anyrange - anyrange → anyrange 计算两个范围的差值。第二个范围不能包含在第一个范围中，否则差值将不是一个范围。

# 范围函数

- lower ( anyrange )→ anyelement 提取范围的下限（如果范围为空或没有下限， NULL ）。 lower(numrange(1.1,2.2)) → 1.1
- upper ( anyrange )→ anyelement 提取范围的上限（如果范围为空或没有上限， NULL ）。
- isempty ( anyrange ) → boolean 该范围是否为空？
- lower_inc ( anyrange ) → boolean 该范围的下限是否包含下限？
- upper_inc ( anyrange ) → boolean 该范围的上限是否包含上限值？
