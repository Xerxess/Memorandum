# 设置返回函数

https://www.postgresql.org/docs/17/functions-srf.html

- 此类函数中最常用的是序列生成函数

```sql
SELECT * FROM generate_series(2,4);
 generate_series
-----------------
               2
               3
               4

SELECT * FROM generate_series('2008-03-01 00:00'::timestamp,
                              '2008-03-04 12:00', '10 hours');
   generate_series
---------------------
 2008-03-01 00:00:00
 2008-03-01 10:00:00
 2008-03-01 20:00:00
 2008-03-02 06:00:00
 2008-03-02 16:00:00
 2008-03-03 02:00:00
 2008-03-03 12:00:00
 2008-03-03 22:00:00
 2008-03-04 08:00:00
(9 rows)
```
