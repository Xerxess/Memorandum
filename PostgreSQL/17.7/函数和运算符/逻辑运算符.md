# 逻辑运算符

```
boolean AND boolean → boolean 
boolean OR boolean → boolean
NOT boolean → boolean
```

### 逻辑运算符真值表

#### AND 和 OR 运算符

| a | b | a AND b | a OR b |
|---|---|---------|--------|
| TRUE | TRUE | TRUE | TRUE |
| TRUE | FALSE | FALSE | TRUE |
| TRUE | NULL | NULL | TRUE |
| FALSE | FALSE | FALSE | FALSE |
| FALSE | NULL | FALSE | NULL |
| NULL | NULL | NULL | NULL |

#### NOT 运算符

| a | NOT a |
|---|-------|
| TRUE | FALSE |
| FALSE | TRUE |
| NULL | NULL |
