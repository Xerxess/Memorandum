# 比较函数和运算符

## 比较运算符

| 运算符 | 描述 |
|--------|------|
| `datatype < datatype → boolean` | 小于 |
| `datatype > datatype → boolean` | 大于 |
| `datatype <= datatype → boolean` | 小于或等于 |
| `datatype >= datatype → boolean` | 大于或等于 |
| `datatype = datatype → boolean` | 等于 |
| `datatype <> datatype → boolean` | 不等于 |
| `datatype != datatype → boolean` | 不等于 |

## 比较谓词

| 谓词 | 描述 | 示例 |
|------|------|------|
| `datatype BETWEEN datatype AND datatype → boolean` | 在之间（包含范围端点） | `2 BETWEEN 1 AND 3` → `t`<br>`2 BETWEEN 3 AND 1` → `f` |
| `datatype NOT BETWEEN datatype AND datatype → boolean` | 不在之间（`BETWEEN`的否定） | `2 NOT BETWEEN 1 AND 3` → `f` |
| `datatype BETWEEN SYMMETRIC datatype AND datatype → boolean` | 在之间，在排序端点值之后 | `2 BETWEEN SYMMETRIC 3 AND 1` → `t` |
| `datatype NOT BETWEEN SYMMETRIC datatype AND datatype → boolean` | 不在之间，在排序端点值之后 | `2 NOT BETWEEN SYMMETRIC 3 AND 1` → `f` |
| `datatype IS DISTINCT FROM datatype → boolean` | 不等于，将null视为可比较值 | `1 IS DISTINCT FROM NULL` → `t`<br>`NULL IS DISTINCT FROM NULL` → `f` |
| `datatype IS NOT DISTINCT FROM datatype → boolean` | 等于，将null视为可比较值 | `1 IS NOT DISTINCT FROM NULL` → `f`<br>`NULL IS NOT DISTINCT FROM NULL` → `t` |
| `datatype IS NULL → boolean` | 测试值是否为null | `1.5 IS NULL` → `f` |
| `datatype IS NOT NULL → boolean` | 测试值是否不为null | `'null' IS NOT NULL` → `t` |
| `datatype ISNULL → boolean` | 测试值是否为null（非标准语法） | |
| `datatype NOTNULL → boolean` | 测试值是否不为null（非标准语法） | |
| `boolean IS TRUE → boolean` | 测试布尔表达式是否产生真 | `true IS TRUE` → `t`<br>`NULL::boolean IS TRUE` → `f` |
| `boolean IS NOT TRUE → boolean` | 测试布尔表达式是否产生假或未知 | `true IS NOT TRUE` → `f`<br>`NULL::boolean IS NOT TRUE` → `t` |
| `boolean IS FALSE → boolean` | 测试布尔表达式是否产生假 | `true IS FALSE` → `f`<br>`NULL::boolean IS FALSE` → `f` |
| `boolean IS NOT FALSE → boolean` | 测试布尔表达式是否产生真或未知 | `true IS NOT FALSE` → `t`<br>`NULL::boolean IS NOT FALSE` → `t` |
| `boolean IS UNKNOWN → boolean` | 测试布尔表达式是否产生未知 | `true IS UNKNOWN` → `f`<br>`NULL::boolean IS UNKNOWN` → `t` |
| `boolean IS NOT UNKNOWN → boolean` | 测试布尔表达式是否产生真或假 | `true IS NOT UNKNOWN` → `t`<br>`NULL::boolean IS NOT UNKNOWN` → `f` |
