# 采用位运算描述状态

* 可通过位运算的规律对多个状态进行合并

# 需要用到位运算符

* |(加)
* ^(减)
* &(比较)

# 定义

```js
    const type1=1<<0
    const type2=1<<1
    const type3=1<<2
    const type4=1<<3

    // 组合
    const type5=type1|type2|type3|type4

    // 判断
    console.log(type5&type1===type1)
    console.log(type5&type2===type2)
    console.log(type5&type3===type3)
    console.log(type5&type4===type4)

    // 剔除
    console.log(type5^type4===(type1|type2|type3))

```

# 注意

* 对框架可以使用
* 对于场景业务不要使用，可读性为0
