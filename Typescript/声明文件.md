# 声明文件

https://github.com/DefinitelyTyped/DefinitelyTyped

## 发布

声明文件发布到 npm 的主要方法有两种:

- 与您的 npm 软件包捆绑在一起
- 在 npm 上发布到@types

```json
// package.json
{
  "name": "awesome",
  "author": "Vandelay Industries",
  "version": "1.0.0",
  "main": "./lib/main.js",
  //   请注意，该"typings"字段与相同"types"，也可以使用
  "types": "./lib/main.d.ts"
}
```
