# 预声明标识符

<https://pkg.go.dev/builtin>

## 内置函数

func append(slice []Type, elems ...Type) []Type 内置函数 `append` 用于向切片末尾添加元素。
func cap(v Type) int cap 内置函数根据 v 的类型返回 v 的容量
func clear[T ~[]Type | ~map[Type]Type1](t T) 内置函数 `clear` 用于清除映射和切片。
func close(c chan<- Type) `close` 内置函数用于关闭一个通道，该通道必须是双向通道或仅发送通道。
func complex(r, i FloatType) ComplexType 内置函数由两个浮点值构造一个复数值。
func copy(dst, src []Type) int 将源切片中的元素复制到目标切片中
func delete(m map[Type]Type1, key Type) 从map表中删除具有指定键 (m[key]) 的元素
func imag(c ComplexType) FloatType 内置函数 imag 返回复数 c 的虚部
func len(v Type) int 内置函数根据 v 的类型返回 v 的长度
func make(t Type, size ...IntegerType) Type 内置函数 `make` 用于分配并初始化一个 `slice`、`map` 或 `chan`（仅限这三种类型）的对象。
func max[T cmp.Ordered](x T, y ...T) T 内置函数返回固定数量的 cmp.Ordered 类型参数中的最大值
func min[T cmp.Ordered](x T, y ...T) T 内置函数返回指定数量的 cmp.Ordered 类型参数中的最小值
func new(Type) *Type 新的内置函数用于分配内存。第一个参数是类型，而不是值，返回值是指向新分配的该类型零值的指针。
func panic(v any) 内置函数会停止当前 goroutine 的正常执行
func print(args ...Type) 内置函数会以特定于实现的方式格式化其参数，并将结果写入标准错误输出。
func println(args ...Type) 内置函数会以特定于实现的方式格式化其参数，并将结果写入标准错误输出。
func real(c ComplexType) FloatType 内置函数 real 返回复数 c 的实部
func recover() any 内置函数允许程序管理发生 panic 的 goroutine 的行为
