# Go语言

- 通用编程语言，专为系统编程设计
- 强类型语言，具有垃圾回收机制
- 支持显式的并发编程
- 语法简洁，便于解析，便于IDE等自动工具分析
- Go 1.25（2025年8月12日发布）

## 常规

- 分号自动插入

## 关键字

关键字（保留字，不能用作标识符）：
break, default, func, interface, select, case, defer, go, map, struct, chan, else, goto, package, switch, const, fallthrough, if, range, type, continue, for, import, return, var

## 整数字面量

- 前缀：0b/0B（二进制）, 0/0o/0O（八进制）, 0x/0X（十六进制）
- 单独0被视为十进制零
- 可使用下划线分隔数字增强可读性

## 浮点数字面量

- 表示浮点数常量
- 十进制形式：整数部分、小数点、小数部分、指数部分（e/E）
- 十六进制形式：0x/0X前缀，尾数、指数部分（p/P）
- 十六进制浮点数的指数是2的幂次（十进制是10的幂次）

## 注释

- 行注释：//开头，到行尾结束
- 块注释：/*开始，*/结束

```go
// 行注释 - 常规用法
/*
  块注释 - 多用于包文档
*/

// 文档注释示例
// Package fmt implements formatted I/O with functions analogous to C's printf and scanf.
package fmt

// 函数文档注释
// NextInt returns the next integer in the scan and the position after it.
func NextInt(b []byte, pos int) (value, nextPos int) {
    // 函数实现
}
```

## 类型系统

- bool 布尔类型，true/false
- uint8, uint16, uint32, uint64 无符号整数（8/16/32/64位）
- int8, int16, int32, int64 有符号整数（8/16/32/64位）
- float32, float64  IEEE 754浮点数（32/64位）
- complex64, complex128 复数（32/64位实部和虚部）
- uint, int 实现相关大小（32或64位）
- uintptr 一个足够大的无符号整数，可以存储指针值的未解释位
- byte uint8别名
- rune int32别名，表示Unicode代码点

## 常量和变量

### 常量

```go

// 基础常量声明
const pi = 3.14159              // 无类型常量
const height int = 180          // 有类型常量

// 批量常量声明
const (
    statusOK   = 200            // 无类型整数常量
    statusNot  = 404            // 无类型整数常量
    msg        = "Not Found"    // 无类型字符串常量
)

// iota常量生成器应用
const (
    a = iota  // 0
    b         // 1（自动继承a的值）
    c         // 2
    d = 5     // 5（重置）
    e         // 5（继续继承）
    f = iota  // 4（重新使用iota）
    g         // 5
)

// 使用iota实现位掩码
const (
    flagRead   = 1 << iota  // 1
    flagWrite               // 2
    flagExecute             // 4
)
```

### 变量

```go
// 标准变量声明
var age int                    // 声明未赋值，默认为0
var name string = "Tom"        // 声明并赋值
var (
    count  int
    price  float64 = 99.99
    active bool
)

// 批量赋值
var x, y, z int
x, y = 1, 2                    // 交换值
x, y = y, x                    // 交换变量

// 短变量声明（函数内使用）
message := "Hello, Go!"        // 自动推断类型
number := 42                   // 推断为int
PI := 3.14159                  // 推断为float64

// 多变量短声明
a, b := 10, 20                 // 声明并赋值
a, c := 30, 40                 // a已存在（重新赋值），c是新变量

// 空白标识符使用
_, err := someFunc()           // 忽略不需要的返回值
_, _ = age, name               // 编译器忽略未使用的变量
```

## 控制结构

```go
// 基础if语句
if x > 0 {
    return y
}

// 带有初始化语句的if
if err := file.Chmod(0664); err != nil {
    log.Print(err)
    return err
}


// 省略else的写法（常见错误处理模式）
f, err := os.Open(name)
if err != nil {
    return err
}
codeUsing(f)


// 链式错误处理
f, err := os.Open(name)
if err != nil {
    return err
}
d, err := f.Stat()
if err != nil {
    f.Close()
    return err
}
codeUsing(f, d)
```
