# 基础模块

## 入口与输出

### fmt

- 格式化 I/O
- 打印日志 (Print, Println, Printf)，读取输入 (Scan, Scanln)，字符串格式化 (Sprintf)。
- 支持类似 C 语言的占位符，如 %d (整数), %s (字符串), %v (任意类型)

## 模版引擎

### text/template

- 用于生成文本输出（如配置文件、邮件内容、代码生成）
- 语法：{{ .FieldName }}, {{ range }}, {{ if }}。

### html/template

- 专门用于生成 HTML
- 安全特性：它内置了上下文感知的转义机制，自动防止 XSS 攻击（例如，在 JavaScript 上下文中转义规则与 HTML body 不同）。写 Web 应用务必用这个，不要用 text/template 生成 HTML。

### io

- 提供了基本的 I/O 原语（接口）
- Reader（读）, Writer（写）, Closer（关闭）。
- 它是所有数据流（文件、网络请求、内存缓冲）的抽象基础。只要你实现了 io.Reader，你就能用标准库里的工具读取你的数据。

### bufio

- 带缓冲的 I/O
- 包装 io.Reader 或 io.Writer，减少直接读写系统调用的次数，提升性能（例如按行读取文本）

## 文本处理

### strings

- 操作 UTF-8 字符串
- 常用函数 Contains (包含), HasPrefix/HasSuffix (前后缀), Split/Join (分割/连接), Replace (替换), ToLower/ToUpper (大小写)

### strconv

- 字符串与基本类型之间的转换
- strconv.Atoi(s): 字符串转整型（ASCII to int）。
- strconv.Itoa(i): 整型转字符串（int to ASCII）
- strconv.ParseBool(s): 解析布尔值（支持 "1", "t", "T", "true", "0", "f", "F", "false"）
- strconv.ParseInt(s, base, bitSize): 按进制（base）和位数（bitSize）解析整数
- strconv.ParseFloat(s, bitSize): 解析浮点数
- strconv.FormatBool(b): 布尔值转字符串
- strconv.FormatInt(i, base): 将整数按指定进制转为字符串
- strconv.FormatFloat(f, fmt, prec, bitSize): 格式化浮点数，可控制精度和格式（如 'f' 或 'e'）

### regexp

- 正则表达式支持

### unicode

- 处理 Unicode 码点、判断字符属性（是否是数字、是否是汉字等

## 网络与 Web

### net

- 底层网络通信
- 包含：net/tcp, net/udp, net/ip 等
- net.Dial (作为客户端连接), net.Listen (作为服务端监听端口)。

### net/http

- HTTP 客户端和服务端实现
- http.HandleFunc, http.ListenAndServe。几行代码就能起一个 Web 服务
- http.Get, http.Post
- ResponseWriter, Request 对象

## 数据编码

### encoding/json

- JSON 序列化与反序列化
- 序列化 json.Marshal(v any) ([]byte, error)。它将变量 v 转为紧凑的 JSON 格式。
- 反序列化 json.Unmarshal(data []byte, v any) error。它将 JSON 字节解码到变量 v 中（通常是结构体指针）。
- 格式化输出 json.MarshalIndent(v, prefix, indent) 可生成带缩进的、易读的 JSON。 

### encoding/xml、encoding/csv

- 处理对应格式的数据

## 并发控制

### sync

- 提供低级并发原语

### context

- 管理请求的生命周期、截止时间、取消信号

## 系统交互

### os

- 操作系统接口

### os/exec

- 运行外部命令
- exec.Command("ls", "-l").Output() 在 Go 程序中执行 Shell 命令

### path/filepath

- 处理操作系统相关的文件路径（处理斜杠 / 或 \ 的区别）

## 时间与数学

### time

- 时间处理和测量
- 核心：time.Now(), time.Sleep(), timeTicker (定时器), time.After。
- 格式化：Go 的时间格式化很特别，是用特定的时间点来布局，例如 2006-01-02 15:04:05

### math

- 数学常数和函数。
- Max, Min, Abs, Sqrt, Sin, Pi, E 等。

## 反射

### reflect

- 在运行时检查类型信息、修改变量值
- 很多标准库（如 json, fmt）底层都依赖它。虽然强大，但通常建议少用，因为会影响代码可读性和性能。
