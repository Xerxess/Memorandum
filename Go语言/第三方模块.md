# 第三方模块

## 工具库

### samber/lo

<https://pkg.go.dev/github.com/samber/lo>

这是一个基于 Go 1.18+ 泛型的实用工具库，它简化了切片、映射、字符串、通道和函数的操作。它提供了数十种便捷的方法，可以简化常见的编码任务并提高代码可读性。在某些方面，它可能与 Lodash 类似。

## Web 开发与 API

### Gin

目前最流行的轻量级 Web 框架，以其类似 Martini 的 API 和极高的路由性能（基于 httprouter）著称。

### Fiber

基于极致性能的 fasthttp 构建，语法风格极度接近 Node.js 的 Express，非常适合追求极致并发的场景。

### Echo

在极简主义与功能丰富度之间取得了极佳平衡，拥有优秀的中间件支持。

## 数据库与 ORM

### go-redis

功能完整、类型安全的 Redis 客户端，支持集群、哨兵模式和流（Streams）。

## 系统工具与配置

### Cobra  

构建命令行工具（CLI）的事实标准，Docker 和 Kubernetes 均使用它开发。

### Viper

配置管理利器，支
持从 JSON、YAML、环境变量等多种来源动态读取配置，通常与 Cobra 配合使用。

### Zap

Uber 开源的高性能、结构化日志库，专门为追求极致速度的日志记录场景优化。
