
# 指针

- 在 Go 语言中，指针（Pointer）是一个简单但核心的概念。
- Go 的指针设计依然坚持 “受限且安全” 的原则：它允许你直接引用内存，但不允许像 C 语言那样进行危险的指针运算。

## 指针的本质

- 定义：指针是一个变量，它存储的是另一个变量的 内存地址。
- 符号
  - & (取地址符)：用于获取一个变量的内存地址。
  - \* (解引用符)：用于访问指针指向的那个地址里存储的具体数值。

```go
package main

import "fmt"

func main() {
    x := 10
    p := &x // p 现在是指向 x 的指针（类型为 *int）

    fmt.Println("x 的值:", x)   // 10
    fmt.Println("x 的地址:", p) // 类似 0xc000012088
    fmt.Println("通过 p 访问 x:", *p) // 10

    *p = 20 // 修改指针指向的内容，x 也会随之改变
    fmt.Println("修改后的 x:", x) // 20
}
```
