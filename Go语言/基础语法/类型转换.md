# 类型转换

## 基本类型转换

- 数值之间的转换最为常见，语法为 Type(expression)。
- 

```go
var i int = 100
var f float64 = float64(i) // int 转 float64
var u uint32 = uint32(f)   // float64 转 uint32（会丢失小数部分）

// 即使在 64 位系统上 int 等于 int64，也要显式转
var i64 int64 = 500
// i = i64             // 编译报错
i = int(i64)           // 编译通过

// 字符串与切片转换
s := "Hello"
b := []byte(s) // 字符串转字节切片 (发生内存拷贝)
s2 := string(b) // 字节切片转字符串 (发生内存拷贝)

// String 与 []rune：用于处理中文字符索引
s := "Go语言"
r := []rune(s) // 结果为 [71 111 35821 35328]
```
