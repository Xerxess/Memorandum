# GraphQL

<https://graphql.org/learn/introduction/>

GraphQL 是一种用于 API 的查询语言，也是一个服务器端运行时环境，用于使用您为数据定义的类型系统来执行查询。GraphQL 规范于 2015 年开源，此后已被多种编程语言实现。GraphQL 不依赖于任何特定的数据库或存储引擎——它由您现有的代码和数据提供支持。

## GraphQL 精要

- 定义：GraphQL 是一种面向 API 的查询语言和服务端运行时，围绕强类型 Schema 执行查询、变更和订阅。客户端声明需要的数据形状，服务端只返回对应字段。
- 关键点（基于官方文档概念）：
  - 强类型 Schema（SDL）、
  - 单端点、
  - 多操作类型（query/mutation/subscription）、
  - 字段级解析（resolvers）、
  - 内省、
  - 自描述、
  - 部分成功（data+errors）、
  - 变量/片段/指令。
- 传输规范：
  - GraphQL over HTTP 规范推荐响应使用 Accept/Content-Type: application/graphql-response+json；
  - 查询可用 GET;
  - 变更建议 POST；
  - 请求体包含 query、variables、operationName。
