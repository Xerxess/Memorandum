# Tailwind CSS v4.0

- Tailwind CSS v4.0 专为 Safari 16.4+、Chrome 111+ 和 Firefox 128+ 而设计
- 需要 Node.js 20 或更高版本

## 重大更改

- PostCSS 插件位于专用的 @tailwindcss/postcss 包中

```ts
// postcss.config.mjs
export default {
  plugins: {
    "@tailwindcss/postcss": {},
  },
};
```

- 删除了@tailwind 指令

```css
/* @tailwind base;
@tailwind components;
@tailwind utilities; */
@import "tailwindcss";

/* 默认 @import "tailwindcss"； 注入的内容如下 */
@layer theme, base, components, utilities;
/* 默认主题 */
@import "tailwindcss/theme.css" layer(theme); 
/* 样式重置 */
@import "tailwindcss/preflight.css" layer(base);
/* 公用 */
@import "tailwindcss/utilities.css" layer(utilities);
```
